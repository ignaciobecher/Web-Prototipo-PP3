Bienvenidos al sistema de Gestión de Cantina

Nuestra aplicación permite una gestión eficiente y sencilla de la cantina de FUPa. Gracias a su versión web y móvil, podrás acceder a las funcionalidades tanto en el escritorio como en tu dispositivo móvil, todo desarrollado con tecnología .NET Core para la web y .NET MAUI para dispositivos móviles.

Servicios

Ofrecemos diversas funcionalidades para optimizar la gestión de tu cantina:

  • Control de consumos.
  • Gestión de pedidos.
  • Integración con dispositivos móviles.
  • Reportes de pedidos.
